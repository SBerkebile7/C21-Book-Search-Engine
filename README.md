# C21-Book-Search-Engine

=======

## Purpose
A website that allows users to search for books using keywords. Users are able to log in and save books for viewing later, should they choose to do so.

## Built With
* JS
* MongoDB / Mongoose
* Express
* React
* Node
* Heroku
